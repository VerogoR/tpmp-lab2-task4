# Overview

Программа «Абитуриент»: чтение данных абитуриентов из текстового файла (фамилия, имя, отчество, пол, национальность, дата рождения, адрес, оценки по ЦТ, проходной балл), просмотр содержимого файла, фильтрация по проходному баллу >= 225 и сохранение результата в новый текстовый файл. Реализована по модели КИС (клиент — интерфейс — сервер). Исходники в `src/`, заголовки в `include/`, сборка через `make`.

# Usage

Сборка: `make`. Запуск: `./applicants [входной_файл] [выходной_файл]`. По умолчанию входной файл — `applicants.txt`, выходной — `result.txt`. Для просмотра содержимого файла: `./applicants --view [файл]`. Пример с тестовыми данными: `cp tests/applicants.txt applicants.txt && ./applicants applicants.txt result.txt`.

# Additional Notes

CI (GitHub Actions) при push и pull_request выполняет `make`, запуск с тестовым файлом `tests/applicants.txt` и проверку того, что в результат попадают только абитуриенты с проходным баллом >= 225. Контрольные расчёты и тесты — в `docs/report.md`.
